# 📚 Comics Villa

Comics Villa is a frontend practice project built using **HTML, CSS, and Media Queries**.  
The main focus of this project was to improve my understanding of **mobile responsiveness** and frontend layout behavior.

## 🚀 Project Focus
For this build, I specifically practiced designing for a **391px mobile viewport size** to understand:
- How layouts behave on smaller screens
- Common responsive design issues
- Fixing overflow and alignment problems using CSS
- Improving UI structure step by step

## ✨ Features
- ✅ Clean and simple UI
- ✅ Mobile-focused responsive layout
- ✅ CSS Media Queries
- ✅ Structured HTML design
- ✅ Beginner-friendly frontend project
- ✅ Deployed using Netlify

## 🧠 What I Learned
While building this project, I improved my skills in:
- Writing cleaner HTML structure
- Styling reusable components with CSS
- Mobile responsiveness techniques
- Debugging layout breaks
- Understanding viewport behavior
- Deploying frontend projects online

## 🛠️ Tech Stack
- HTML
- CSS
- Media Queries
